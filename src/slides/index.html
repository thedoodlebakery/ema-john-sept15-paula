JSX

Why JSX

  • Basic Html powered by Javascript
  • Describe how UI should look like
  • Put markup and logic at the sameplaces
  • It's not mandatory but everyone uses it
  • Allows usage of all JavaScript power

JSX Summary

  • Valid HTML and custom components as tag
  • Extra attributes to pass as props
  • Put all dynamic part inside {}
  • Events camel cases onCLick
  • attributes camel cases tabIndex, colSpan
  • className instead of class attribute
  • Add key while looping array to create elements

Props

Props

  • Send data to the child component
  • Pass data it by attribute
  • Values set by attribute={}
  • Can set default value outside of the receiving Component
  • children is automatically added to pass child element
  • props is readonly

State

State

  • Declared inside constructor
  • Set initial value while declaring
  • Don't set value directly
  • Use this.setState() and pass key-value
  • only available inside the component where it declared
  • State update is asynchronous
  • this.setState()can take a function as well

State vs Props

Description State Props
Purpose Hold changing value Pass data to child
Where used Declared Component Child Component
Change in Component No Yes
Set Default value constructor Receiving component
